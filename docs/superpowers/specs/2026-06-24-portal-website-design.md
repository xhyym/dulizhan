# 独立站门户网站设计文档

## 项目概述

为 Indie Station 家具桌凳品牌打造面向欧美市场的独立站门户网站，采用极简北欧风格，强调产品展示和用户体验。

## 技术架构

- **前端**: Next.js 14 + App Router + SSR
- **后端**: 现有 Java Spring Boot API (`/api/portal/`)
- **样式**: Tailwind CSS
- **部署**: Vercel 或自建 Node 服务

### 渲染策略

| 路径 | 页面 | 渲染方式 | 说明 |
|------|------|---------|------|
| `/` | 首页 | SSR | SEO 重要 |
| `/products` | 商品列表 | SSR + 客户端筛选 | SEO + 交互体验 |
| `/products/[id]` | 商品详情 | SSR | SEO 最重要 |
| `/about` | 关于我们 | SSR | 品牌介绍 |
| `/cart` | 购物车 | CSR | 需登录 |
| `/inquiry` | 询盘确认 | CSR | 需登录 |
| `/account` | 用户中心 | CSR | 需登录 |
| `/account/orders` | 询盘记录 | CSR | 需登录 |

## 设计规范

### 风格定义

- **主色调**: 黑白灰，强调产品本身
- **留白**: 大量留白，呼吸感
- **字体**: 无衬线字体（Inter / Noto Sans）
- **图片**: 高质量产品图，场景图

### Header（导航栏）

**初始状态**: 完全透明，白色文字，浮在 Hero Banner 上
**滚动状态**: 白色半透明毛玻璃背景，深色文字

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo]    Home  Shop  About  Contact    🔍  🛒(3)  👤Login │
└─────────────────────────────────────────────────────────────┘
```

- Logo 左侧
- 导航链接居中偏左
- 搜索、购物车（带数量角标）、登录/用户头像右侧
- 移动端：汉堡菜单

### Footer（页脚）

## API 依赖

门户端已实现的 API：

| 模块 | 接口 | 鉴权 |
|------|------|------|
| 认证 | `POST /api/portal/auth/login` | 公开 |
| 认证 | `GET /api/portal/auth/me` | 需登录 |
| 认证 | `POST /api/portal/auth/logout` | 需登录 |
| 商品 | `GET /api/portal/categories` | 公开 |
| 商品 | `GET /api/portal/products` | 公开 |
| 商品 | `GET /api/portal/products/{id}` | 公开 |
| 商品 | `GET /api/portal/products/new` | 公开 |
| 购物车 | `GET /api/portal/cart` | 需登录 |
| 购物车 | `POST /api/portal/cart/items` | 需登录 |
| 购物车 | `PUT /api/portal/cart/items/{id}` | 需登录 |
| 购物车 | `DELETE /api/portal/cart/items/{id}` | 需登录 |
| 购物车 | `DELETE /api/portal/cart` | 需登录 |
| 询盘 | `POST /api/portal/inquiries` | 需登录 |
| 询盘 | `GET /api/portal/inquiries` | 需登录 |
| 询盘 | `GET /api/portal/inquiries/{id}` | 需登录 |
| 配置 | `GET /api/portal/site-config` | 公开 |
| 访客 | `POST /api/portal/visit` | 公开 |
